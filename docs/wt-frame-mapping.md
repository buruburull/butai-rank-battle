## 1. ブレード系（STRIKER）

### 1-1. 弧月 (Kogetsu) → **Crescent（クレセント）**

**WT原作での特徴**:
万能型の標準ブレード。攻撃力が高く、リーチも長い。オプションとして「旋空（センクウ）」で斬撃を飛ばせる。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | NETHERITE_SWORD |
| 基本ダメージ | 8 |
| ダメージ倍率 | 1.3倍 |
| エーテルコスト | 0（常時） |
| 攻撃速度 | 普通（バニラ準拠） |

**特殊効果**:

- **通常攻撃**: 近接ダメージ1.3倍。最も安定した近接火力
- **旋空（スキル発動）**: スニーク+右クリックで「斬撃波」を発射
  - 射程: 10ブロック直線
  - ダメージ: 基本ダメージ×1.5（= 15.6）
  - エーテルコスト: 80/回
  - クールダウン: 10秒
  - MC実装: 不可視のArmorStandを高速で直進させ、接触した敵にダメージ。パーティクル（SWEEP_ATTACK）で軌跡表示
  - **弧月最大の切り札。ハイコスト・ハイリターン**

**バランス意図**: 近接最高火力。旋空は強力だが80エーテルは重い。使いどころの判断が腕の見せ所

### 1-2. スコーピオン (Scorpion) → **Fang（ファング）**

**WT原作での特徴**:
変形自在の軽量ブレード。攻撃力は低いが攻撃速度が速い。体のどこからでも刃を出せる。「マンティス」（二刀流強化）や「ブランチブレード」（多方向同時攻撃）が可能。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | GOLDEN_SWORD |
| 基本ダメージ | 5 |
| ダメージ倍率 | 1.0倍 |
| エーテルコスト | 0（常時） |
| 攻撃速度 | 高速（Attack Speed属性+30%） |

**特殊効果**:

- **背面攻撃ボーナス**: 背後からの攻撃で1.5倍（合計2.25倍）
- **マンティス（二刀流）**: スニーク+右クリックで5秒間「マンティスモード」発動
  - 攻撃時に追加ヒット（1回の攻撃で2回分のダメージ判定）
  - 攻撃速度さらに+20%
  - エーテルコスト: 50/発動 + 10/秒（持続）
  - 攻撃をやめると自動解除
  - MC実装: EntityDamageByEntityイベントで同フレーム内に追加ダメージ適用
- **ブランチブレード**: パッシブ。攻撃した敵の半径2ブロック以内に別の敵がいる場合、25%の確率で周囲の敵にもダメージ（本来のダメージの50%）
  - MC実装: 攻撃イベント時にNearbyEntitiesを検索

**バランス意図**: 単体火力は弧月に劣るが、速度と背面攻撃で暗殺性能が高い。マンティスはエーテルを食うが爆発力あり

---

### 1-3. レイガスト (Raygust) → **Bastion（バスティオン）**

**WT原作での特徴**:
攻防一体の重量ブレード。シールドモードに変形できる。「スラスター」オプションで突進攻撃が可能。攻撃力は低いが防御力が高い。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | IRON_SWORD（攻撃時） / SHIELD（防御時） |
| 基本ダメージ | 6 |
| ダメージ倍率 | 1.0倍 |
| エーテルコスト | 0（攻撃） / 5/秒（シールド） |
| 攻撃速度 | やや遅い（Attack Speed -15%） |

**特殊効果**:

- **シールドモード切替**: 右クリックでシールドモード ON/OFF
  - シールド中: 被ダメ60%カット、攻撃力半減、移動速度-10%
  - エーテルコスト: 5/秒
- **スラスター突進**: シールドモード中にスニーク+右クリック
  - 前方12ブロックを高速突進
  - 突進中に接触した敵に ダメージ10 + ノックバック大
  - エーテルコスト: 70/回
  - クールダウン: 15秒
  - MC実装: プレイヤーのvelocityを前方ベクトルに大きく設定。移動中のtickごとにNearbyEntitiesチェック
  - 突進中は被ダメ80%カット（シールド構えたまま突っ込むイメージ）

**バランス意図**: 攻撃力は最低だが生存力が高い。スラスターで奇襲もできる。チーム戦での前衛・壁役

## 2. 射撃系（GUNNER）

### ガンナー vs シューターの区別

WTでは「ガンナー」（銃型、精度安定、弾道固定）と「シューター」（弾を自由操作、テクニカル）が分かれている。

**BRBでの再現**:
- 同じフレームでも **通常撃ち（ガンナーモード）** と **操作撃ち（シューターモード）** を切り替え可能
- **ガンナーモード（デフォルト）**: 左クリック発射。弾道固定、精度安定
- **シューターモード（スニーク中）**: スニーク+左クリック発射。弾を自由操作できるが、コスト1.5倍
- シューターモードの操作方法はフレームごとに異なる（後述）

---

### 2-1. アステロイド (Asteroid) → **Pulse（パルス）**

**WT原作での特徴**:
通常弾。まっすぐ飛ぶ。威力が高い。シンプルだが最も汎用的。集中させて「ギムレット」（貫通弾）にもできる。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | BOW |
| 基本ダメージ | 5 |
| ダメージ倍率 | 1.0倍 |
| エーテルコスト | 10/発（ガンナー）/ 15/発（シューター） |
| 弾速 | 高速（velocity × 1.5） |

**特殊効果**:

- **ガンナーモード**: 弓のフルチャージ射撃。弾速が他の射撃フレームの1.5倍。まっすぐ飛ぶ
- **シューターモード（スニーク発射）**: 弾を3発同時に扇状に発射（各ダメージ60%）
  - エーテルコスト: 15 × 3 = 45/回
  - MC実装: Arrow×3を微妙に角度をずらして同時発射
- **フルチャージボーナス**: フルチャージ（1秒ホールド）で威力1.3倍
- **集中弾（ギムレット）**: 後述（合成弾セクション）

**バランス意図**: 弾速が最速で当てやすい。シンプルだが安定感がある。初心者から上級者まで使える

---

### 2-2. メテオラ (Meteora) → **Nova（ノヴァ）**

**WT原作での特徴**:
炸裂弾。着弾時に爆発する。空中で任意に爆破可能。分割して散弾のように使える。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | CROSSBOW |
| 基本ダメージ | 6（直撃） / 4（爆風） |
| ダメージ倍率 | 1.0倍 |
| エーテルコスト | 20/発（ガンナー）/ 30/発（シューター） |
| 爆発半径 | 4.0ブロック |

**特殊効果**:

- **ガンナーモード**: 着弾時に爆発（半径4.0、ダメージ4）。地形破壊あり（試合後復元）
- **シューターモード（スニーク発射）**: 発射後、もう一度右クリックで **空中起爆**
  - 起爆しなければ着弾で通常爆発
  - MC実装: 飛翔中のArrowをトラッキングし、右クリック検知で即座にExplosion生成
- **分裂モード**: スニーク+左クリックで「分裂弾」発射
  - 着弾点で4方向に小爆発（各ダメージ2、半径2.0）
  - エーテルコスト: 30/回
  - MC実装: 着弾時にSmallFireballを4方向に生成
- **ノックバック強化**: 爆風のノックバック距離が通常の2倍

**バランス意図**: 面制圧に特化。直撃火力はそこそこだが、閉所での制圧力は全フレーム随一。地形破壊で遮蔽物を壊せる

---

### 2-3. バイパー (Viper) → **Frost（フロスト）**

**WT原作での特徴**:
変化弾。発射前に弾道をプログラムできる。壁の裏に回り込む軌道、フェイントをかける軌道など自由自在。高等技術。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | BOW |
| 基本ダメージ | 4 |
| ダメージ倍率 | 1.0倍 |
| エーテルコスト | 15/発（ガンナー）/ 25/発（シューター） |
| 弾速 | やや遅い（velocity × 0.8） |

**特殊効果**:

- **ガンナーモード**: 発射方向に直進するが、**照準先のブロック/エンティティに向かって緩やかに曲がる**
  - 曲がり具合: 1tickあたり弾道を5度ずつ目標方向へ修正
  - 射程: 25ブロック
  - MC実装: Arrow発射後、毎tickでProjectileを目標座標に向けてvelocity修正
- **シューターモード（スニーク発射）**: **2段階曲射**
  1. スニーク+発射で弾を撃つ（一時停止ポイントに向かって飛ぶ）
  2. 弾が中間地点に到達したら、その時点の照準方向に再び曲がる
  - 壁の裏に回り込む射撃が可能
  - MC実装: Arrowに2つのウェイポイントを設定。第1ウェイポイント到達後、プレイヤーの現在視線方向で第2ウェイポイントを再計算
- **命中効果**: Slowness II（3秒）+ Weakness I（2秒）
  - 当たれば敵の機動力と火力を同時に削る

**バランス意図**: 直進弾より当てにくいが、遮蔽物越しに撃てる唯一のガンナーフレーム。シューターモードは上級者向け

---

### 2-4. ハウンド (Hound) → **Seeker（シーカー）**

**WT原作での特徴**:
誘導弾。自動で敵を追尾する。威力は通常弾より低い。複数同時発射で包囲するように追尾させるのが定石。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | TRIDENT（Loyalty風の追尾演出） |
| 基本ダメージ | 4 |
| ダメージ倍率 | 1.0倍 |
| エーテルコスト | 15/発（ガンナー）/ 20/発（シューター） |
| 追尾射程 | 20ブロック |

**特殊効果**:

- **ガンナーモード**: 単発ホーミング弾。発射方向の最も近い敵に自動追尾
  - 追尾強度: 1tickあたり10度修正
  - 追尾射程: 20ブロック（範囲外の敵には追尾しない）
  - MC実装: Trident投擲後、毎tickでNearestEnemyに向けてvelocity修正
- **シューターモード（スニーク発射）**: **3連射ホーミング**
  - 0.3秒間隔で3発連続発射。全弾が同一ターゲットを追尾
  - エーテルコスト: 20 × 3 = 60/回
  - 3発同時に別ターゲットを追尾はしない（最初のターゲット固定）
- **包囲射撃**: スニーク+右クリックで「分散追尾」モード
  - 3発を左・上・右の3方向に分けて発射し、全弾が同一ターゲットに向かう
  - 異なる角度から飛来するのでシールドで全弾防ぐのが困難
  - エーテルコスト: 60/回
  - MC実装: 3つのTridentを120度ずつずらして発射し、全て同じターゲットに追尾

**バランス意図**: 命中率は最高だが単発火力は最低。数で押す戦い方。エーテル消費が激しいので長期戦は苦手

## 3. 狙撃系（MARKSMAN）

### 3-1. イーグレット (Egret) → **Falcon（ファルコン）**

**WT原作での特徴**:
バランス型スナイパーライフル。射程・威力・弾速のバランスが良い。最も使用者が多い。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | BOW |
| 基本ダメージ | 7 |
| ダメージ倍率 | 2.5倍（フルチャージ時） |
| エーテルコスト | 25/発 |
| チャージ時間 | 2.0秒 |

**特殊効果**:

- **チャージシステム**: 弓を引いている時間に応じてダメージ増加
  - 0〜0.5秒: 1.0倍（通常）
  - 0.5〜1.0秒: 1.5倍
  - 1.0〜2.0秒: 2.0倍
  - 2.0秒以降: 2.5倍（最大）= ダメージ17.5
- **チャージ完了表示**: 2.0秒到達時にパーティクル（END_ROD）とサウンド（BLOCK_NOTE_BLOCK_PLING）で通知
  - 味方にも敵にも見える（駆け引き要素）
- **射程強化**: 矢の重力落下を50%軽減（遠距離でも弾道が安定）
  - MC実装: 毎tickでArrowのY速度に上方向の微小ベクトルを加算

**バランス意図**: 狙撃の基本形。チャージが完了すれば確実に大ダメージだが、チャージ中は無防備。安定した中距離〜遠距離戦闘

---

### 3-2. ライトニング (Lightning) → **Volt（ヴォルト）**

**WT原作での特徴**:
高速弾・貫通力特化。弾速が最も速い。威力は控えめだが、複数の敵やシールドを貫通できる。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | BOW |
| 基本ダメージ | 6 |
| ダメージ倍率 | 1.2倍 |
| エーテルコスト | 20/発 |
| 弾速 | 最速（velocity × 2.0） |
| 貫通 | 最大3体 |

**特殊効果**:

- **超高速弾**: 弾速が全フレーム最速。重力落下もほぼなし
  - MC実装: Arrow速度を通常の2倍に設定。setGravity(false)に近い挙動
- **貫通**: 矢が敵を貫通し、最大3体にヒット
  - 貫通ボーナス: 1体目1.2倍、2体目1.4倍、3体目1.6倍
  - MC実装: ProjectileHitイベントでArrowをキャンセルせず、同じベクトルで新しいArrowを生成
- **チャージ不要**: 引きの長さに関係なくダメージ一定（速射が可能）
  - MC実装: BowShootイベントでforce値に関係なくダメージを固定

**バランス意図**: チャージ不要の速射型。単発火力はFalcon/Zenithに劣るが、手数と貫通で差別化。チーム戦で特に強い

### 3-3. アイビス (Ibis) → **Zenith（ゼニス）**

**WT原作での特徴**:
最大火力スナイパーライフル。チャージ時間が長いが一撃必殺級の威力。発射時の閃光で位置がバレる。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | BOW |
| 基本ダメージ | 8 |
| ダメージ倍率 | 3.0倍（フルチャージ時） |
| エーテルコスト | 35/発 |
| チャージ時間 | 3.0秒 |

**特殊効果**:

- **チャージシステム**: 3.0秒フルチャージで3.0倍 = ダメージ24
  - 0〜1.0秒: 1.0倍
  - 1.0〜2.0秒: 1.5倍
  - 2.0〜3.0秒: 2.5倍
  - 3.0秒以降: 3.0倍（最大）
- **位置露出**: 発射時に自分にGlowing（発光）効果を2秒付与
  - 全敵から位置が丸見えになる
  - MC実装: BowShootイベントで射手にPotionEffect(GLOWING, 40tick)付与
- **衝撃波**: 命中時にノックバック大（通常の3倍）
  - 仮に即死しなくても吹き飛ばされてポジションを失う
- **弾道の光跡**: 矢の軌道にパーティクル（FLASH + END_ROD）を表示
  - どこから撃たれたかが一目瞭然
  - MC実装: 毎tickでArrow位置にパーティクルをスポーン

**バランス意図**: 全フレーム最大火力。フルチャージすればほぼ一撃だが、3秒の隙+位置バレというハイリスク。ロマン砲

## 4. オプショントリガー（SUPPORT）

### 4-1. シールド (Shield) → **Barrier（バリア）**

**WT原作での特徴**:
空間にバリアを展開する防御トリガー。視線方向の任意の位置に配置できる。形状も変えられる（集中→強度UP、拡大→範囲広い）。最も基本的な防御手段。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | SHIELD |
| エーテルコスト | 25/使用 |
| クールダウン | 3秒 |
| 持続時間 | 2.0秒 |

**特殊効果**:

- **シールド展開**: 右クリックで視線方向5ブロック先に **3×3の透明バリア** を展開
  - 矢・Trident・その他Projectileを遮断（ProjectileHitイベントでキャンセル）
  - プレイヤーの通過は可能（当たり判定なし）
  - MC実装: 指定位置にBARRIERブロックを3×3で設置。パーティクル（BLOCK_CRACK + GLASS）で視覚化。2秒後に撤去
- **集中シールド**: スニーク+右クリックで「1×1の高強度シールド」
  - Zenithの一撃も1回だけ防げる（通常シールドは威力15以上で貫通される）
  - エーテルコスト: 40/使用
  - 持続時間: 1.0秒
- **フルガード**: 自分の周囲全方向にシールド展開（360度防御）
  - エーテルコスト: 80/使用
  - 持続時間: 1.5秒
  - クールダウン: 20秒
  - 全方向からの射撃を防ぐが、コストが非常に重い

**バランス意図**: 最も汎用的な防御手段。低コストで頻繁に使える。集中/フルガードは緊急時の切り札

---

### 4-2. エスクード (Escudo) → **Rampart（ランパート）**

**WT原作での特徴**:
地面から巨大な壁を生やす防御トリガー。シールドと違い物理的な壁なので射線を完全に遮断する。設置後は消えるまで残り続ける。

**MC実装仕様**:

| 項目 | 値 |
|---|---|
| MCアイテム | GOLD_BLOCK |
| エーテルコスト | 50/使用 |
| クールダウン | 8秒 |
| 持続時間 | 8秒 |

**特殊効果**:

- **壁生成**: 右クリックで視線先の地面から **幅5×高さ4の壁** を生成
  - ブロック: STONE_BRICKS（耐久度あり）
  - 各ブロックの耐久度: 30ダメージで破壊可能
  - MC実装: ターゲットブロックの上にsetBlock()で壁を建設。BlockChangeListenerで試合後に復元
- **壁の特性**:
  - 敵味方問わず通行不可（物理的な壁）
  - 射撃を完全遮断
  - Novaの爆発で破壊可能（爆発で耐久度大幅減少）
  - 8秒後に自動消滅（パーティクルで崩壊演出）
- **足場利用**: 壁の上に乗ることが可能。高所ポジションの即席確保に使える

## 7. ガンナー vs シューター モードまとめ

射撃系フレーム（GUNNER）は2つのモードを持つ:

| モード | 操作 | 特徴 | コスト倍率 |
|---|---|---|---|
| ガンナーモード | 通常射撃 | 弾道固定、安定、シンプル | ×1.0 |
| シューターモード | スニーク+射撃 | 弾操作可能、テクニカル | ×1.5 |

シューターモードの操作はフレームごとに異なる:
