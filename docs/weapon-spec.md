# 武器仕様書

## メイン武器

### 近接武器

| WT名称 | BRB名称 | 仕様 |
|--------|---------|------|
| 弧月（こげつ） | クレセント | 基礎攻撃力が高く、シンプルな刀 |
| スコーピオン | ファング | 基礎攻撃力は低いが、軽いため機動力が長所 |
| レイガスト | バスティオン | 基礎攻撃力は中程度、武器自体をシールドに切り替えることが可能 |

### 射撃武器

| WT名称 | BRB名称 | 仕様 |
|--------|---------|------|
| アステロイド | パルス | 射撃コストは低くダメージも低い |
| メテオラ | ノヴァ | 射撃コストは高く、ダメージの高い球が分散し着弾時に爆発する |
| バイパー | フロスト | 射撃コスト中程度、ダメージは低く、敵に球がヒット時スロウを付与する |
| ハウンド | シーカー | 射撃コストは中程度、ダメージは低く球が標的をホーミングする |
| イーグレット | ファルコン | 射撃コスト中、チャージ時間2秒で2倍のダメージになる |
| ライトニング | ヴォルト | 射撃コスト少、敵を貫通して速射が可能 |
| アイビス | ゼニス | 射撃コスト高、チャージ3秒でダメージ3倍 |

## サブ武器

| WT名称 | BRB名称 | 仕様 |
|--------|---------|------|
| メテオラサブ | ブラスト | コスト高、爆弾を投擲し、着弾箇所を爆破し地形破壊も可能 |
| カメレオン | クローク | 1秒ごとに仕様コストがかかる、使用中は透明化し攻撃などを行うと透明化が解除される |
| テレポート | ワープ | コスト高、向いている方向にTPをする |
| エスクード | ヴァント | コスト中、自身の正面に3×3の壁を生成する |
| グラスホッパー | リープ | コスト中、自身の向いている方向にダッシュをする。空中での使用も可能 |
| 強化レーダー | トレーサー | コスト高、周囲64m以内の敵を発光させる |
